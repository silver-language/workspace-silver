
Features
========

Main features
-------------
The main things I want, and why this is really here:

* homoiconicty
* readability
* meta programming






Silver-data
-----------

Speculative:
- References/pointers
- derived types


Silver-language
---------------
Main:
- array manipulation
- code blocks
- functions


Speculative:
- mutability / immutabilty
- privacy / visibility (cf Java)
- scope restriction
- lifetimes / borrows (cf rust)

Possible exclusions:

- classical OO - classes, inheritance etc
- closures
