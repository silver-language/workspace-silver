Document Format
===============


```yaml
myDoc: Document
	heading: H1 `This is the heading of the document`

	introduction: Paragraph
		`lorum ipsum`
		`dolor sit amet`

	H2 `Requirements`
	UnorderedList
		Paragraph
			`item 1`
			``
		`item 2`
		`item 3`

	H2 `Instructions`
	OrderedList
		`do this`
		`then this`
		`finally this`

	script: Code
		`10 print "Hello world"`
		`20 goto 10`

	references: UnorderedList
		Link `http://localhost/help`
myDoc end
```
