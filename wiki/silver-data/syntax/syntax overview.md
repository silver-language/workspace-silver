Syntax Overview
===============


The overall form:

	name: Type value


Fundamental value types are array and string


Simple assignment

	myValue : SomeType foobar


Array assignnment

	myArray: SomeArrayType
		value1
		value2
		...


Array values

	myArray.value1
	myArray.value2
	...


Metadata

	name..value

	eg

	myValue..type
	myValue..name
