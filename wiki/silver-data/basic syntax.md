Basic syntax
============

Silver-data main features
-------------------------


The overall form:

	name: type value


Fundamental value types are array and string


Simple assignment

	myValue : someType foobar


Array assignnment

	myArray: someArrayType
		value1
		value2
		...


Array values

	myArray.value1
	myArray.value2
	...


Metadata

	name..value

	eg

	myValue..type
	myValue..name
